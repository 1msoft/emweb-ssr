# emweb-ssr

一个基于`next.js`的`ssr`项目

## 快速开始

1. `clone`项目

```bash
git clone https://github.com/1msoft/emweb-ssr.git
```

2. 安装依赖

```bash
cd emweb-ssr
npm install
```

3. 运行项目

```bash
npm run dev
```

4. 访问页面

```bash
http://localhost:3000
```

## 项目部署

1. 项目打包

```bash
npm run build
```

2. 运行项目

```bash
npm run start
```
